# Cape-Able Heroes – Challenge Cards (v1)

**Purpose:** Printable cards used during WonderLab sessions. Each card connects a hero identity to a specific challenge, a reflection question, and a completion acknowledgment.  
**Status:** v1 — 5 themed examples ready to use. Add themes after pilot testing.

---

## Card Template

Every Challenge Card uses this structure:

```
┌─────────────────────────────────────────────┐
│  ⬡  CAPE-ABLE HEROES WONDERLAB              │
│  [THEME TITLE]                               │
├─────────────────────────────────────────────┤
│  HERO NAME: _____________________________   │
│  MY STRENGTH: ___________________________   │
├─────────────────────────────────────────────┤
│  YOUR MISSION:                               │
│  [One sentence: story-framed challenge]      │
│                                              │
│  YOUR TASK:                                  │
│  [Clear, specific action]                    │
│                                              │
│  FIRST STEP:                                 │
│  [The single smallest move to begin]         │
├─────────────────────────────────────────────┤
│  REFLECTION QUESTION:                        │
│  [One question to complete after the task]   │
├─────────────────────────────────────────────┤
│  ✓ MISSION COMPLETE:                         │
│  I tried this today: ___________________     │
│  My strength helped me by: _____________     │
└─────────────────────────────────────────────┘
```

---

## Print Layout Notes (v1 — No Design Software Required)

- Print 2 cards per page (landscape A4 or Letter)
- Minimum font size: 14pt for child-read sections, 12pt for facilitator-facing fields
- Use a simple border (1–2px) — no complex design needed for v1
- Leave generous blank space for writing (children's handwriting is large)
- Cards are filled in during Phase 2 (Hero Identity) and Phase 4 (Reflection) of the session

For digital versions: create in Google Docs or Canva using the template above. Export as PDF for printing.

Every hero story starts with one problem and one person willing to try. Today, you are writing that story.

## Adding New Themes

To create a new Challenge Card, use this checklist:

- [ ] Theme name (1–2 words)
- [ ] Mission intro (1 sentence — story-framed, ties to the theme world)
- [ ] Task (one clear hands-on activity, completable in 15–25 min)
- [ ] First Step (the single smallest move — lowers the barrier to starting)
- [ ] Materials needed (list)
- [ ] Reflection Question (1 question focused on *what they tried*, not how it turned out)
- [ ] Completion fields (same on every card)

Submit new themes as a SAGA proposal or add directly to this file after pilot testing.

## Hero Card (Post-Session Version)

The Hero Card is completed during Phase 4 (Reflection) and given to the child to keep.  
This is different from the Challenge Card — it is a **record of what happened**, not an instruction.
