# Cape-Able Heroes

This folder contains the development work for **Cape-Able Heroes**, one of the core offerings within the BYDT ecosystem.

## Purpose

Cape-Able Heroes helps children feel capable, recognized, and brave enough to step forward by treating identity as architecture. It is built on the belief that children should not have to prove they are capable before they are treated as capable.

## Current Focus

- Developing a repeatable, sellable WonderLab experience
- Creating supporting tools (Challenge Cards, take-home kits, observation methods)
- Testing through pilot and paid sessions
- Building toward a scalable offer ladder
