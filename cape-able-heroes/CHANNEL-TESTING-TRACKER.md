# Cape-Able Heroes – Channel Testing Tracker (v1)

**Purpose:** Track outreach, conversion, and revenue across all buyer channels during Phase 3. This is the primary audit artifact for go-to-market learning.  
**Status:** v1 — Active for Phase 3. Run a SAGA Analyze cycle after 5–10 contacts per channel.

---

## How to Use This Tracker

1. Log every outreach contact the moment it is sent — not after you hear back.
2. Update the status as it moves through the pipeline.
3. After 5–10 contacts with any channel, check the conversion rate and update the Channel Summary below.
4. When you see a pattern — run a SAGA Analyze cycle and file a proposal if significant.

**Status codes:**
- `Sent` — outreach made, no response yet
- `Responded` — any response (positive or not)
- `Booking call` — they want to learn more / have a conversation
- `Booked` — session scheduled and confirmed
- `Ran` — session completed
- `No response` — followed up once, no reply after 7+ days
- `Not interested` — explicitly declined

## SAGA Analyze Triggers

Run a SAGA Analyze cycle and file a proposal when **any** of the following occur:

- [ ] 10 total contacts logged with less than 20% response rate — **something is wrong with the message**
- [ ] 5+ contacts in one channel with 0 bookings — **this channel may not be right for Phase 3**
- [ ] One channel has a conversion rate 2x higher than all others — **double down here**
- [ ] A new buyer type appears that is not in the tracker — **add it and assess**
- [ ] Total revenue from Phase 3 activations reaches $2,500 — **SAGA review for Phase 4 readiness**
- [ ] Any single activation generates $1,000+ — **document the full pathway as a repeatable model**
