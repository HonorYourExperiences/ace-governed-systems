## First Session Trigger

When both of these are true, Session 1 is imminent:

- [ ] At least one person has said "yes" in some form (verbal, written, confirmed booking)
- [ ] A date and location are confirmed

At that point, move to `SESSION-LOG.md` and prepare the observation tool.

---

## If Nothing Is Moving

If 7 days have passed since the first outreach was sent with no response:

1. Run the Way Through self-check in `FOUNDER-WEEKLY-REFLECTION.md` first.
2. Then ask: Is the message not landing, or am I not sending it?
3. If the message is the issue — revise one template and try again with 2 new contacts.
4. If sending is the issue — that is a founder signal, not a product signal. Name it.
