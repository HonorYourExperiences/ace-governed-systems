# Cape-Able Heroes – Facilitator Observation Tool (v1)

**Use:** During Phase 3 (Challenge Activity) and Phase 4 (Reflection)  
**Purpose:** Capture real field evidence — not scores, not assessments. Evidence of what happened.  
**Status:** v1 — use as-is for pilot sessions. Refine based on what you actually notice.

---

## How to Use This Tool

You are not evaluating children. You are **witnessing** them and capturing evidence.

- Use one sheet per session (group-level observations)
- Add individual notes only when something specific and meaningful happens
- Write what you *see* and *hear* — not what you interpret or feel
- If you have a co-facilitator, one person runs the room and one observes

## What to Do With This Sheet

1. Keep all completed sheets — they are field evidence and audit artifacts
2. After 3 sessions: review patterns across sheets and create a SAGA proposal to update the session format
3. Use the data to refine facilitator guidance, challenge difficulty, and materials
4. File completed sheets in: `cape-able-heroes/field-evidence/` (create folder after first session)
